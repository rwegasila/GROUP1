SEMESTER II COURSES AND THEIR APPLICATIONS

STUDY COURSES

COMPUTER NETWORKS FUNDAMENTALS

This Course enhances students have a gain a foundational understanding of computer networking. This comprehensive module covers topics such as TCP/IP, OSI models, networking devices, and practical applications. By the end of the course, students can describe networks using a five-layer model, understand standard protocols, troubleshoot networks, and explore cloud computing

MULTIMEDIA TECHNOLOGIES

This course has been designed to train learners in producing and utilizing multimedia technology for education. It covers concepts, skills, and attitudes related to multimedia. The course is divided into three modules, each comprising five units1. Additionally, the Media and Information Literacy Grade 12 Alternative Delivery Mode (ADM) Module provides opportunities for guided and independent learning in multimedia information and media2. Another related course, Introduction to Multimedia Systems, delves into image/video coding, streaming multimedia.

WEB APPLICATION

This course imparts foundational knowledge and practical skills for constructing dynamic and responsive web applications. Topics include HTML, CSS, JavaScript, and back-end development. Learners gain expertise in database design, security, scalability, and user experience, enabling them to create modern full-stack web applications

BASICS OF MOBILE APPLICATION

In this course,learners acquire essential skills for creating innovative mobile applications. The curriculum covers user interface design, cross-platform development, and coding for both Android and iOS environments. Topics include using Java for Android apps, Swift for iOS apps, and even building cross-platform apps with React Native1. Additionally, learners explore tools like Android Studio and gain insights into operating systems and mobile app creation

CARRIER APPLICATIONS

COMPUTER NETWORKS CARRIERS

NETWORK ADMINSTRATOR
The network field is expansive, encompassing diverse roles from cybersecurity to network administration, with salaries averaging $80,000 monthly.
CYBERSECURITY PROFESSIONALS.
Cybersecurity professionals safeguard networks, earning around $100,000 monthly, their expertise crucial in protecting against digital threats and breaches.

MULTIMEDIA FIELD APPLICATION

GRAPHICS DESIGNER
In graphic design, creativity meets communication, shaping visual identities. Salaries range from $3,000 to $6,000 monthly, depending on experience.
ANIMATION ARTISTS.
In animation, artists bring stories to life. Salaries vary but average $4,000 to $8,000 monthly, reflecting skill level and project scope .

WEB APPLICATION CARRIERS

WEB DEVELOPERS.
Web application development merges coding and design, creating interactive online experiences. Salaries range from $5,000 to $10,000 monthly.
WEB PROFESSIONALS
In web development, professionals build and maintain websites. Salaries vary, averaging $4,000 to $8,000 monthly, influenced by experience and specialization.

MOBILE APPLICATION FIELD APPLICATION

MOBILE APP DEVELOPERS
In mobile app development, engineers craft software for smartphones. Salaries range from $6,000 to $12,000 monthly, reflecting skills and project complexity.
DATABASE ADMINSTRATOR
In database administration, specialists manage and optimize data storage. Salaries typically range from $5,000 to $10,000 monthly, depending on expertise.
